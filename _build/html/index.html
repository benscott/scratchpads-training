Scratchpads¶

Scratchpads are an easy and effective way to present your data online.

Whether you just want to publish on the web, construct a database or assemble a formal manuscript, Scratchpads are able to help!

Useful links¶

Scratchpads platform
Find information about Scratchpads, browse through sites and apply for a new Scratchpad site http://scratchpads.eu
Issues queue
Report an issue, ask your questions or find answers or to request a new feature https://github.com/NaturalHistoryMuseum/scratchpads2/issues
Sandbox
Experiment with all Scratchpads features through this fully functional test site that resets every 6 hours. http://sandbox.scratchpads.eu
